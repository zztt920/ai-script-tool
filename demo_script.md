# AI 剧本创作工具 — 演示视频脚本

> 录制时长建议：8-12 分钟 | 旁白语言：中文
> 录制工具推荐：OBS Studio（免费开源）或 Windows 自带录屏（Win+Alt+R）

---

## 第一部分：项目概述（30 秒）

**【画面】** 打开 VS Code 展示项目目录结构，依次高亮 domain/ service/ adapter/ api/ cli/ db/ 目录

**【旁白】**
"AI 剧本创作工具是一个将小说自动转换为结构化剧本的开源项目。它基于 Python + FastAPI 构建，采用分层架构，支持中英双语输出。让我们从启动服务开始，逐步体验它的全部功能。"

---

## 第二部分：启动服务（40 秒）

**【画面】** 终端中输入命令：

```bash
pip install -r requirements.txt
cp .env.example .env
# 编辑 .env 填入 DeepSeek API Key
python -m uvicorn api.main:app --host 0.0.0.0 --port 8000
```

**【旁白】**
"首先安装依赖并配置环境变量。项目使用 DeepSeek API 驱动 AI 分析，需要在 .env 文件中填入 API Key。启动服务后，访问 localhost:8000 即可打开主界面。"

**【画面】** 展示终端日志输出 —— 数据库初始化、限流中间件启用、服务就绪

---

## 第三部分：主界面浏览（1 分钟）

**【画面】** 浏览器打开 http://localhost:8000/ ，依次点击顶部导航栏：
- **创作页** —— 小说输入 + 风格选择 + AI 配置
- **作品页** —— 已完成/进行中的任务列表
- **风格页** —— 展示全部 10 种风格模板及分幕结构
- **关于页** —— 项目信息

**【旁白】**
"主界面采用书籍出版风格设计。顶部是导航栏，切换到'创作'页面可以看到章节输入区、文件上传区，以及风格配置面板。项目内置了 10 种风格模板——悬疑、甜宠、热血、古装、仙侠、科幻等，每种风格有独特的分幕叙事结构。"

**【画面】** 点击右上角语言切换按钮，中英文切换演示

**【旁白】**
"右上角的语言切换支持中文和英文两种界面语言，并且切换后剧本输出也会对应改变——中文时所有 YAML 字段名也会翻译为中文。"

---

## 第四部分：提交转换任务（1.5 分钟）

**【画面】** 
1. 点击"载入示例"按钮，自动填充 3 章示例小说内容
2. 填写原著标题"长安十二时辰"，作者"马伯庸"
3. 选择风格"古装"，模式保持"精细模式"
4. 选择语言"中文"
5. 点击"开始创作"按钮

**【旁白】**
"点击载入示例可以快速体验完整流程。这里我们使用古装风格模板，它会将剧本分为入局、博弈、翻覆、定鼎四幕。点击开始创作后，后端会启动 9 步转换流水线——从加载文本、AI 分析人物、提取场景、润色对白，到构建分幕结构、YAML 生成、Schema 校验和质量审核。"

**【画面】** 展示任务卡片出现，进度条实时更新，当前步骤文字变化

**【旁白】**
"右侧会实时显示任务进度，包括当前执行到哪一步，已处理多少章节和场景。进度每 3 秒自动轮询更新。"

---

## 第五部分：书本翻页阅读器（1.5 分钟）

**【画面】**
1. 切换到"作品"页
2. 点击已完成任务的"在线查看"按钮
3. 展示书本翻页阅读器

**【旁白】**
"任务完成后，点击在线查看进入书本翻页式阅读器。这是项目的一大亮点——采用类似真实纸质书的翻页动画，支持鼠标拖拽翻页、点击页边翻页，以及键盘左右箭头翻页。阅读器完整展示剧本结构——封面、故事梗概、人物介绍、按场次/按拍节排列的内容，全部以中文呈现。"

**【画面】** 
- 翻到封面页
- 翻到故事梗概
- 翻到人物介绍页
- 翻到具体场景内容
- 演示翻页动画拖拽效果

---

## 第六部分：剧本下载与导出（1 分钟）

**【画面】**
1. 在作品页点击"下载 YAML"按钮，展示下载的 YAML 文件内容（含中文 key）
2. 切换到 Swagger UI（http://localhost:8000/docs），找到 `GET /api/v1/tasks/{task_id}/script.json` 端点
3. 点击"Try it out"，输入 task_id，展示 JSON 格式导出结果

**【旁白】**
"完成的剧本支持多种格式下载。YAML 格式可以直接下载，JSON 格式通过 API 端点导出。如果选择了中文输出，所有 YAML 字段名——如'改编说明'、'主题关键词'、'场景列表'——都会自动翻译为中文。你还可以通过 Swagger UI 文档页面在线尝试所有 API 端点。"

---

## 第七部分：Swagger API 文档（1 分钟）

**【画面】** 打开 http://localhost:8000/docs

**【旁白】**
"项目内置了完整的 OpenAPI 3.0 文档，通过 Swagger UI 呈现。可以看到所有 API 端点的完整列表——"
"转换提交 POST /api/v1/convert"
"任务列表 GET /api/v1/tasks"
"任务详情 GET /api/v1/tasks/{task_id}"
"剧本下载 GET /api/v1/tasks/{task_id}/script"

"全文搜索 GET /api/v1/tasks/search"
"批量删除 POST /api/v1/tasks/batch-delete"
"还有认证相关端点和 Schema 校验功能。"

**【画面】** 依次点击几个端点的 "Try it out" 按钮，展示请求/响应

---

## 第八部分：用户认证（40 秒）

**【画面】**
1. 在 Swagger UI 中找到 `/auth/register`，注册新用户
2. 使用 `/auth/login` 登录，获取 JWT Token
3. 粘贴 Token 到页面顶部的 Authorize 按钮
4. 调用 `/auth/me` 查看当前用户信息

**【旁白】**
"系统还集成了完整的 JWT 用户认证。注册后登录获取 Token，后续请求附上 Token 即可验证身份。密码采用 bcrypt 风格哈希存储，确保安全。"

---
